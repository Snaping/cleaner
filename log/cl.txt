## 已实现的功能

### 🔍 **7大垃圾分类，30+子分类**

| 大类 | 子分类 |
|------|--------|
| **📁 系统临时文件** | Windows临时目录、用户临时目录、预读取文件、缩略图缓存、内存转储、系统日志、WinSxS备份 |
| **💾 应用数据缓存** | AppData临时文件、漫游缓存、崩溃报告、浏览器缓存(Chrome/Edge/Firefox)、Electron应用缓存、媒体软件缓存 |
| **📦 包管理器缓存** | NPM、Yarn、Pip、NuGet、Maven、Gradle、Cargo、Go模块、Pnpm |
| **💻 IDE和编辑器缓存** | VSCode缓存、废弃VSCode扩展(旧版本检测)、Visual Studio缓存、JetBrains IDE缓存 |
| **🔄 系统更新垃圾** | Windows更新缓存、Windows.old旧系统备份、更新日志、传递优化文件 |
| **🗑️ 应用程序垃圾** | 应用日志、ProgramData缓存、卸载残留(空目录检测) |
| **📋 备份与碎片** | 废弃备份(.bak/.old等)、安装临时文件、下载缓存 |

### ✨ **核心功能**

1. **深度扫描** - 智能遍历磁盘，精准识别各类垃圾文件
2. **分类显示** - 按大类/小类树状展示，显示文件路径、大小、来源、修改时间
3. **进度反馈** - 扫描和清理过程实时进度条 + 当前操作显示
4. **权限提升** - 使用 `sudo-prompt` 自动请求管理员权限删除系统保护文件
5. **灵活选择** - 支持全选/取消/按分类选择/单个文件选择
6. **一键清理** - 快捷清理所有检测到的垃圾文件
7. **结果统计** - 显示清理数量、释放空间、失败文件列表

### 🛡️ **安全机制**

- 文件删除前确认弹窗
- 标记需要管理员权限的文件(🔒图标)
- 清理失败不中断整体流程，单独记录失败项
- 自定义无框窗口，避免误操作
